# laravel-test

## Установка
### Клонируем репозиторий:   
```
git clone https://github.com/Grunov/laravel-test.git
```

### Устанавливаем зависимости:
```
composer install
```

## Настройка

1. Переимновываем файл .env.example в .env
2. Генерируем ключ
```
php artisan key:generate
```
3. Создаем storage link
